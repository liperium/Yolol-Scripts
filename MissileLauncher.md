# Missile Launcher

Shoots missiles one by one, and makes sure you lock everything up too. 

Reset button lets you refill the launcher.

## Usage
### Preparation

Make sure your missiles are built correcly and that they are locked from the fuel tank.

### Naming
```
-- Button --
    Name = ShootM
    Type = 1
```
```
-- Button --
    Name = ResetM
    Type = 1
```
```
-- Progress Bar 12x24 --
    Name = MissilesLeft
    MinValue=0
    MaxValue=9
```
