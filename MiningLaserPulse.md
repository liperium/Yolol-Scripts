# Mining Laser Pulsing

Pulses two laser groups for energy reduction.

Turns on collector when pressing the button. **( "//" before ":collect=1" to remove this feature | line 3)**

## Usage

### Naming
```
-- Button --
    Name = Laser
    Type = 1
```
```
-- Mining Lasers --
 - Group 1 -
    Name = LaserG1

 - Group 2 -
    Name = LaserG1
```
